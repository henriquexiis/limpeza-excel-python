# Limpeza de Planilhas Excel com Python

Script em Python para limpar, padronizar e organizar planilhas Excel.

## O que o script faz
- Remove linhas vazias
- Padroniza textos
- Preenche valores faltantes
- Gera arquivo final com data
- Exibe relatório no terminal

## Como usar
1. Coloque sua planilha como `input.xlsx`
2. Execute:
   python limpeza_excel.py
3. O arquivo final será gerado automaticamente
